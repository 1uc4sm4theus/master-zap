## Packages
vis-network | Interactive network graph visualization
vis-data | Data handling for vis-network
date-fns | Date formatting for messages
lucide-react | Icons

## Notes
- `vis-network` is used for the "Modo Grafo" feature to render the connections.
- The application simulates a WhatsApp Web interface, using specific colors like #075E54 and #efeae2.
- Data is primarily read-only for educational simulation purposes based on public data.
