**MasterZap** é uma aplicação web que organiza e apresenta **exclusivamente dados públicos** divulgados pela Polícia Federal e pela imprensa brasileira (reportagens de G1, Folha de S.Paulo, Poder360, O TEMPO, Malu Gaspar, Metrópoles etc.).

Inspirado no projeto **jmailarchive** (arquivo de e-mails do Epstein), transforma vazamentos jornalísticos em uma interface familiar de chat estilo WhatsApp, com grafo de conexões, buscas avançadas e visualização clara.

**Aviso importante**  
- 100% baseado em fontes públicas e jornalísticas já divulgadas.FEITO COM RECURSOS MINIMOS: I.A + Galaxy 11  
- Não contém, acessa ou distribui dados privados, hackeados ou não públicos.  
- Projeto neutro de transparência e jornalismo cidadão.

## Features

- Interface de chat mobile-first idêntica ao WhatsApp
- Grafo interativo de conexões entre contatos e grupos(Ainda em Construção)
- Fotos de perfil reais (quando disponíveis em fontes públicas)
- Busca por nomes, números, palavras-chave, datas ou grupos Na barra de pesquisa principal ("A Turma", etc.)
- Atualizações manuais com novos vazamentos públicos
- Design responsivo 

## Tecnologias

- **Frontend**: React(TypeScript)
- **Dados**: Estruturados em JSON a partir de reportagens públicas
- **Hospedagem inicial**: Replit[](https://master-zap--lucasmatheus20.replit.app)
